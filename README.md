# Dwello Landing Page

##  Overview
Dwello is a responsive **real estate landing page** developed during my training period.  
The project was designed to practice modern UI/UX concepts using **Bootstrap** and to integrate **Swiper.js** for interactive property showcases.

##  Tech Stack
- HTML  
- CSS  
- Bootstrap  
- JavaScript  
- Swiper.js  

##  Features
- Fully responsive design across mobile, tablet, and desktop.  
- Clean real estate layout with sections for services, properties, and contact.  
- **Swiper.js** integration for smooth property image sliders.  
- Organized structure: Header, Hero, Services, Properties, Footer.  
- Training-focused design with practical implementation of responsive layouts.

##  Outcomes
- Learned how to build a real estate landing page with Bootstrap.  
- Gained hands-on experience in responsive design.  
- Successfully implemented **Swiper.js** for property showcases.  
- Improved project organization and UI/UX understanding.
